## pkg-backtop Component

### use npm or yarn
- install
```
npm install pkg-backtop

or

yarn add pkg-backtop
```

- use
```
import BackTop from "pkg-backtop";

new BackTop();
```

### use CDN
```
<script src="/backtop.js"></script>
<script>
new BackTop();
</script>
```
